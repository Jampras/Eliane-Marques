# Plano de Otimizacao da Home para Conversao

**Versao:** 1.2  
**Data:** 19/03/2026

## 1. Objetivo

Transformar a home em uma pagina:
- mais clara
- mais direta
- menos fragmentada
- mais consistente entre desktop e mobile
- mais orientada a decisao

## 2. Status atual no branch

- hero reescrito
- narrativa reorganizada
- metodo nomeado
- bloco de autoridade criado
- CTA reforcado
- mobile simplificado para aproximar a leitura do desktop

O que falta:
- validacao visual/comercial final
- fechamento da direcao mobile
- aprovacao antes de publicar

## 3. Problemas que o plano ataca

- hero generico
- excesso de texto passivo
- repeticao de ideias
- pouca clareza entre ofertas
- mobile ainda mais fraco que desktop

## 4. Criterio de pronto

A home so pode ser considerada pronta quando:
- desktop e mobile estiverem coerentes entre si
- a leitura estiver simples e direta
- a oferta principal estiver obvia
- a pagina estiver validada localmente antes de push/deploy
